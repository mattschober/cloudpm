# Matt Schober Cloud Project Management Website

## Deployment Instructions

This website is designed to be deployed to both:
- **mattschober-cloudpm.com**
- **migratewithmatt.com**

### Files Included

1. **index.html** - Main website file (optimized for SEO and AI discovery)
2. **robots.txt** - Allows all search engines and AI crawlers
3. **sitemap.xml** - XML sitemap for search engines
4. **.htaccess** - Apache configuration for redirects and optimization

### Deployment Steps

#### Option 1: FTP/SFTP Upload
1. Connect to your web hosting via FTP/SFTP
2. Upload all files to the root directory (public_html, www, or httpdocs)
3. Ensure file permissions are set correctly:
   - Files: 644
   - Directories: 755

#### Option 2: cPanel File Manager
1. Log into your cPanel
2. Open File Manager
3. Navigate to public_html (or your domain's root directory)
4. Upload all files
5. Extract if uploaded as a zip file

#### Option 3: Git Deployment
1. Create a repository with these files
2. Connect your hosting to the Git repository
3. Deploy automatically on push
